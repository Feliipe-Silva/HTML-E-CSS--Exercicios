Principais Formatações

Negrito / Destaque:

Termo em negrito usando a tag: < b > (não semântica).

Termo em destaque usando a tag: < STRONG > (semântica).

Itálico / Ênfase:

Termo em itálico usando tag: < i > (não semântica).

Termo emitálico usando tag: < em > (semântica).

Marca Texto:

Usano marca texto com a tag: < MARK style="background-color: aqua;" > // para mudar a cor.

Texto GRANDE e Texto Pequeno

Letras Grandes com a tag: < big >

Letras Pequenas com tag: < small >

Texo Deletado

Um texto excluido para idicar que pode ser lido, mas desconsiderado. Com a tag: < del >

Texto Inserido

Um texto inserido para marcar e dizer que foi inserido depois. Tag: < ins >

E também tem a tag: < u > (não semântica).

Texto Sobrescrito

Para adicionar digitos superior. EX: x20+3 com a tag: < sup >

Texto Subscrito

Para adicionar digitos inferior H2O com a tag: < sub >
